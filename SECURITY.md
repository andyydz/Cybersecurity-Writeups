# Security Policy

## Supported Versions

This repository contains educational TryHackMe writeups and learning notes.
There are no versioned releases. The latest content on the `main` branch is actively maintained.

## Reporting a Vulnerability

If you find:
- Incorrect or outdated information
- Sensitive information accidentally included
- A security concern in this repository

Please do one of the following:
1. Open a GitHub Issue.
2. Contact me through GitHub.

I will review the report and respond as soon as possible.

Thank you for helping improve this repository.
