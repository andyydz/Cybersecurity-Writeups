This folder contains my notes and write-up on **Blue Team Fundamentals**, covering the defensive mindset and core responsibilities of a security team focused on protecting, monitoring, and defending an organization's systems.

## Overview

Where offensive security ("red team") focuses on finding and exploiting weaknesses, blue team work is about the opposite side — building defenses, monitoring for threats, and responding when something goes wrong. This room lays the groundwork for that mindset before moving into more specific SOC tooling and workflows.
