This folder contains my notes and write-ups from the **SOC Level 1** learning path on TryHackMe — practical, hands-on rooms simulating the day-to-day work of an entry-level SOC Analyst.

## Overview

Where earlier rooms in this repo cover the theory behind SOC tooling (SIEM, EDR, IDS), this folder is about applying it — triaging real alerts, reporting findings, and using workbooks and lookups to investigate indicators as part of an analyst's actual workflow.

## Writeups

| File | Focus |
|---|---|
| `soc-l1-alert-triage.md` | Triaging alerts — assessing severity, context, and determining true positive vs. false positive |
| `soc-l1-alert-reporting.md` | Writing up and reporting on a triaged alert, including what details matter for escalation |
| `soc-workbook-and-lookup.md` | Using SOC workbooks and IOC lookups to enrich an investigation with additional context |

## SOC Analyst Relevance

This is the most directly applicable folder in the repo to an actual L1 SOC role — alert triage and reporting are the core daily tasks the job is built around.
