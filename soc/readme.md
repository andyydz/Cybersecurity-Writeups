# SOC (Security Operations Center)

This folder contains my notes and write-ups from SOC-related rooms and modules completed on [TryHackMe](https://tryhackme.com/p/andyydz57).

The purpose of these write-ups is to build a strong understanding of Security Operations Center (SOC) concepts, threat detection, log analysis, incident response, and real-world cybersecurity monitoring practices.

SOC operations are one of the most important areas in cybersecurity because they focus on detecting, analyzing, and responding to security threats in real time.
