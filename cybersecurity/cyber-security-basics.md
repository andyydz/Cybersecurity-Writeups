# Introduction to Cyber Security – TryHackMe

## Overview
In this room, I learned the basic concepts of cybersecurity and got an introduction to both offensive and defensive security. The room also explained different career paths in cybersecurity and how security professionals help protect systems, networks, and data from attacks.

This room gave me a beginner-friendly understanding of how the cybersecurity field works and what different security roles involve.

---

## Key Concepts

### Offensive Security Intro
Learned the basics of offensive security and how ethical hackers test systems to identify vulnerabilities before attackers can exploit them.

Topics included:
- Ethical hacking basics
- Penetration testing concepts
- Finding vulnerabilities
- Thinking from an attacker’s perspective

### Defensive Security Intro
Learned how defensive security focuses on protecting systems, monitoring threats, and responding to attacks.

Topics included:
- Security monitoring
- Threat detection
- Incident response basics
- Blue team responsibilities

### Careers in Cyber Security
Explored different career paths available in cybersecurity and learned about the responsibilities of various security roles.

Some roles covered:
- SOC Analyst
- Penetration Tester
- Security Engineer
- Incident Responder
- Threat Analyst
