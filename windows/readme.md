## Why This Matters

Windows is the most common enterprise operating system, and Active Directory sits at the center of most corporate networks — understanding the command line, PowerShell, and AD fundamentals is essential for recognizing both normal administrative activity and the techniques attackers use to persist and move laterally within a Windows environment.
