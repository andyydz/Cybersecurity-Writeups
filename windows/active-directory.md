# Active Directory

## Overview
Active Directory (AD) is a directory service developed by Microsoft for managing users, computers, groups, and policies within a Windows domain environment.

---

# Important Components

## Domain
A centralized environment that manages users and systems.

---

## Domain Controller (DC)
A server that authenticates users and enforces security policies.

---

## Users and Groups
Used for access management and permissions.

---

## Organizational Units (OU)
Containers used to organize users and devices.

---

# Authentication

## Kerberos
Default authentication protocol used in Active Directory.

---

# Common AD Services

- User authentication
- Group Policy management
- Access control
- Centralized administration

---

# Group Policy

Group Policy allows administrators to configure system settings across domain computers.

---

# Common Active Directory Attacks

## Kerberoasting
Extracting service account tickets for password cracking.

---

## Pass-the-Hash
Using NTLM hashes for authentication without knowing passwords.

---

## LDAP Enumeration
Gathering information from Active Directory.
